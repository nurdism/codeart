# codeart
Turns your code into pixels!

![Preview](https://i.imgur.com/3z5Rzau.png)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
